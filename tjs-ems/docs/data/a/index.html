数据预览

查看数据​

勾选需要查看的点位数据，选择查看的时间间隔和时间范围条件，点击查询，即可查看图形和列表数据。

取消勾选图表，不显示图表

点击打印按钮可以生成pdf，或者直接打印。

点击excel按钮可以生成excel列表数据。

保存配置​

tip

保存配置主要可以解决每次都要手动勾选点位和设置条件的问题，并自由组合报表。

将点位和条件选择合适后，您可以将这次报表条件动态保存，方便以后打开能管系统可以直接在【数据报告】中查看。
